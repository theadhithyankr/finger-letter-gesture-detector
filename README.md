## Setup and Run
```bash
git clone https://github.com/theadhithyankr/finger-letter-gesture-detector.git
cd finger-letter-gesture-detector
python -m venv .venv
.\\.venv\\Scripts\\Activate.ps1
pip install -r requirements.txt
python app.py
```
